# Buruma
Just an application to make all the Umamusume wear buruma shorts (bloomers). However, you can also change a few other things like height, bust size, socks, tail, etc.

# Using
You can just download the buruma.exe and run it or if you want to feel leet, run the python 3 code

`python buruma.py`

# Making the exe
I just use the follow command to make the exe after any code changes and requires you into install `pyinstaller` to make it work. I pretty much did as much as I could to have near zero dependencies for this project.

` python -m PyInstaller --onefile --clean --icon=haru.ico buruma.py`
